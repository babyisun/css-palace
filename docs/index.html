CSS Palace

这是一个用纯CSS写的图标库，没有一丝多余代码，简洁而明快。

说明

  • 支持 IE9+，及所有现代浏览器
  • 通过font-size控制图标大小，通过color控制颜色
  • 通过重写样式，可以支持多色图标，并配合css动画使用

使用步骤：

第一步：引入项目下面生成的 fontclass 代码：

<link rel="stylesheet" href="./css-palace.css">

第二步：挑选相应图标并获取类名，应用于页面：

<span class="css-palace cp-xxx"></span>

可以通过重写 "css-palace" 来改变图标全局样式，如大小、颜色。
